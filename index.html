Assetify is our package for Sass and Sprockets, with common libraries we need for our projects at Bitaculous.

Installation

  1. Add Assetify to your Gemfile:

    gem 'bitaculous-assetify', github: 'bitaculous/assetify'

  2. Run bundle to install all dependencies with Bundler.

Usage

Sass / CSS

normalize.css

The most basic option is simply:

@import normalize

which includes the “full package”.

Or import only specific parts:

@import normalize/base
@import normalize/html5_display_definitions
@import normalize/links
@import normalize/text_level_semantics
@import normalize/embedded_content
@import normalize/grouping_content
@import normalize/forms
@import normalize/tables

reset.css

Import reset:

@import reset

Sass Media Queries

Just import Sass Media Queries:

@import media_queries

and you're ready to roll.

Development

Run specs with RSpec

Run rspec.

or via Guard:

$ guard -g spec

See Test Coverage

Run COVERAGE=true rspec.

Run RuboCop

Run rubocop.

To run all specs and RuboCop altogether, run rake.

Bug Reports

Github Issues are used for managing bug reports and feature requests. If you run into issues, please search the issues and submit new problems here.

Versioning

This library aims to adhere to Semantic Versioning 2.0.0. Violations of this scheme should be reported as bugs. Specifically, if a minor or patch version is released that breaks backward compatibility, that version should be immediately yanked and / or a new version should be immediately released that restores compatibility.
