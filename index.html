Assetify is our package for Sprockets, a Rack-based asset packaging system, with common libraries we need for our projects at Bitaculous.

Installation

  1. Add Assetify to your Gemfile:

    group :assets do
  gem 'assetify', github: 'bitaculous/assetify'
end

  2. Run bundle install

Usage

CSS

normalize.css

First, import normalize:

@import normalize

The most basic option is simply:

@include normalize

which includes the “full package”.

Or include only specific parts:

@include base
@include html5-display-definitions
@include links
@include text-level-semantics
@include embedded-content
@include grouping-content
@include forms
@include tables

reset.css

First, import reset:

@import reset

Second, include it:

@include reset

Sass Media Queries

Just import Sass Media Queries:

@import media_queries

and you're ready to roll.

JavaScript

Versioning

This library aims to adhere to Semantic Versioning 2.0.0. Violations of this scheme should be reported as bugs. Specifically, if a minor or patch version is released that breaks backward compatibility, that version should be immediately yanked and / or a new version should be immediately released that restores compatibility.
